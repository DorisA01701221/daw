Cuestionario 1: BD y DBMS


  • ¿Qué es entonces una base de datos - DB? es un conjunto de datos almacenados previamente para después hacer un uso de ellas, es una colección de información, archivos interrelacionados, que puede ser organizada o no. cubriendo todos los datos posibles y dependiendo de la empresa lograr que los datos se repitan o no tanto, al menos.

  • ¿En qué casos es conveniente usar bases de datos? en todos cuanto sea posible, desde puntos de venta, almacenes, producción, comunicación, hasta ocio. Desde algo en lo que se necesite muchos hasta pocos usuarios.

  • ¿Qué es un sistema de gestión de base de datos - DBMS? es la colección de datos cuando son interrelacionados y cuando se accede a ellos por búsqueda o para seguir incrementandolos.

  • Enlista y explica las funciones/responsabilidades que tienen los DBMS:
    • seguridad: no todos los accesos serán para todos los usuarios. Ejemplo: un usuario que es un empleado de una tienda de abarrotes puede registrar una venta pero no puede acceder a ver las estadísticas de ventas semanales, eso lo podría hacer el gerente.

    • administración: se requiere de gran espacio de almacenamiento de acuerdo a la utilidad de cada empresa.

    • seguridad: si la empresa es una tienda en línea y hay pagos.

  • Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.cuando hice una página parare el catálogo de ropa: hubiera hecho un buscador, hacer accesos (login), registrar nuevas ropas, pagos.
